# QuackitoBot 🦆

Um bot interativo para grupos no Telegram.

O QuackitoBot conversa, faz piadas, puxa desafios, roda dado, faz quiz, sorteia cara ou coroa e movimenta grupos parados com uma personalidade debochada.

## Funções

- `/start` — inicia o bot
- `/menu` — mostra o menu interativo
- `/piada` — manda uma piada
- `/desafio` — lança um desafio para o grupo
- `/dado` — rola um dado
- `/caraoucoroa` — sorteia cara ou coroa
- `/quiz` — inicia um quiz
- `/ranking` — mostra um ranking zoeiro
- `/duelo @usuario` — inicia um duelo entre membros

## Tecnologias

- Node.js
- Telegraf
- dotenv
- Telegram Bot API
